CamiXon.indices — Method
function indices(A::AbstractArray{T,N}, a::T...)  where {T,N}

The index (indices) of selected Array elements (default: all elements).

Examples:

A = collect("ahsgh")
indices(A,'h')
1-element Array{Array{Int64,1},1}:
 [2, 5]

indices(A)
4-element Array{Array{Int64,1},1}:
 [1]
 [2, 5]
 [3]
 [4]
